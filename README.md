# Mathe-Macher

A simple browser-based math trainer for practicing basic arithmetic with a streak system.

## Install & Run


**Option 1 — Local**

1. Download or clone the repository

   ```bash
   git clone https://github.com/Maxel8/Mathe-Macher.git
   ```
2. Open `index.html` in your browser.

**Option 2 — Online**
Use GitHub Pages:
https://maxel8.github.io/Mathe-Macher/

Requires a modern browser with JavaScript enabled.

## Roadmap v0.2

- Difficulty levels (easy / medium / hard)  
- Press Enter to submit the answer  
- Select operation types (addition only, multiplication only, etc.)  
- Color feedback (green/red) instead of text only  
- Dark mode

# Mathe-Macher

Ein einfacher Mathe-Trainer im Browser zum Üben der Grundrechenarten mit Streak-System.

## Installation & Start

**Option 1 — Lokal**

1. Repository herunterladen oder klonen
2. `index.html` im Browser öffnen

**Option 2 — Online**
https://maxel8.github.io/Mathe-Macher/

Benötigt einen modernen Browser mit aktiviertem JavaScript.

## Roadmap v0.2

- Schwierigkeitsstufen (leicht / mittel / schwer)  
- Enter-Taste zum Bestätigen der Eingabe  
- Auswahl der Rechenarten (nur +, nur × usw.)  
- Farbfeedback (grün/rot) statt nur Text  
- Dark Mode
